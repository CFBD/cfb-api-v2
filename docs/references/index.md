# References

Last reviewed: 2026-07-06

This directory is for stable reference material that supports implementation:
schema notes, generated artifact instructions, external API notes, and
operational runbooks.

## Current Sources Of Truth

| Area                              | Source                                     | Refresh                                                                    |
| --------------------------------- | ------------------------------------------ | -------------------------------------------------------------------------- |
| Package commands and dependencies | `../package.json`                          | Update with dependency or script changes.                                  |
| TSOA route and OpenAPI generation | `../tsoa.json`                             | Run `pnpm build` after controller or spec changes.                         |
| TypeScript compiler settings      | `../tsconfig.json`                         | Update when build behavior changes.                                        |
| Jest configuration                | `../jest.config.ts`                        | Update when test discovery or transforms change.                           |
| Kysely database types             | `../src/config/types/db.d.ts`              | Run `pnpm build:db` against an appropriate database schema.                |
| Release and deploy workflow       | `../.github/workflows/release.yml`         | Update when release, image, deploy, or client-generation behavior changes. |
| Security scan workflow            | `../.github/workflows/codeql-analysis.yml` | Update when CodeQL language or schedule changes.                           |

## Generated Artifacts

`build/routes` and `build/swagger.json` are generated by TSOA through
`pnpm build`. Treat them as build outputs: update source annotations and config,
then regenerate instead of editing generated files directly.
